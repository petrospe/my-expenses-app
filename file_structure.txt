## 📁 Δομή Αρχείων Εφαρμογής Κοινοχρήστων

Ακολουθεί η δομή αρχείων της εφαρμογής, όπως παρουσιάστηκε προηγουμένως, σε μορφή Markdown:

```
my-expenses-app/
├── public/
│   └── index.html               # Κύριο αρχείο HTML
├── src/
│   ├── components/
│   │   ├── Header.js            # Κεφαλίδα / Πλοήγηση
│   │   ├── Navigation.js        # (Προαιρετικό) Συστατικό πλοήγησης
│   │   └── ExpenseList.js       # Συστατικό για εμφάνιση λίστας εξόδων
│   ├── pages/
│   │   ├── Home.js              # Αρχική Σελίδα (/)
│   │   ├── Expenses.js          # Σελίδα Διαχείρισης Εξόδων (/expenses)
│   │   ├── Tenants.js           # Σελίδα Διαχείρισης Διαμερισμάτων/Ενοίκων (/tenants)
│   │   └── Calculate.js         # Σελίδα Υπολογισμού Κοινοχρήστων (/calculate)
│   ├── data/
│   │   ├── expenses.json        # Δεδομένα εξόδων (ως "βάση δεδομένων")
│   │   └── tenants.json         # Δεδομένα διαμερισμάτων/ενοίκων (ως "βάση δεδομένων")
│   ├── App.js                   # Κύριο συστατικό (Ρύθμιση React Router)
│   ├── index.js                 # Entry point της εφαρμογής
│   └── index.css                # Γενικό CSS (ή άλλα αρχεία styling)
├── package.json                 # Εξαρτήσεις και scripts του Node.js
└── README.md
```
